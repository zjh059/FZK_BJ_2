焊接次数“/40”修改说明

镀金片焊接上位机｜手机离线版

直接答案

界面上的“/40”写在语言资源文件中。如果要改成“/50”，需要把资源内容从 /40 改为 /50。

当前上位机有简体中文、繁体中文、英文三个语言文件。为避免切换语言后又变回 /40，建议三个文件一起修改。

需要修改的文件

语言文件位置
简体中文CNS.xaml约第35行
繁体中文CNT.xaml约第35行
英文English.xaml约第35行

简体中文完整路径

FZK.Application.UI\Language\CNS.xaml

繁体中文完整路径

FZK.Application.UI\Language\CNT.xaml

英文完整路径

FZK.Application.UI\Language\English.xaml

怎样快速定位

  1. 打开上述语言文件。
  2. 按 Ctrl + F。
  3. 搜索：次数上限
  4. 找到下面这行代码。
<system:String x:Key="次数上限">/40</system:String>

例如改成50次

<system:String x:Key="次数上限">/50</system:String>

三个语言文件中的这一行都改成相同数字即可。

界面在哪里使用这个“/40”

FZK.Application.Run\Views\RunView.xaml
  • 视图名称：RunView
  • 位置：焊接次数显示卡片区域
  • 约第132行和第141行
  • 快速搜索：DynamicResource 次数上限
<TextBlock Text="{DynamicResource 次数上限}" ... />

这里不直接写数字，它会自动读取语言文件里 x:Key="次数上限" 对应的文字。

另一个容易混淆的40

FZK.Application.Share\Run\JigConfig.cs
  • 类名：JigConfig
  • 位置：类的属性区域，约第26行
public int ClearCounts { get; set; } = 40;
注意：经过代码检索，ClearCounts 当前没有被实际业务代码使用。只修改这里不会改变界面上的“/40”，也不会形成真正的40次限制。

最后必须分清

  • 只想把界面 /40 改成 /50：修改三个语言文件。
  • 想让设备达到50次后自动停止：不能只改界面文字，还需要检查 PLC 梯形图和上位机是否有次数到限判断。
  • 当前上位机代码：没有发现有效的“达到40次禁止焊接”判断。
